# CarbonNowCLI.jl

## Setup

- Install Julia
- Run the following command in your terminal

```sh
$ cd path/to/this/directory
$ julia --project -e 'using Pkg; Pkg.instantiate()'
```

## Usage

```sh
$ julia --project main.jl path/to/file
```

### Examples

```sh
$ julia --project main.jl Project.toml
```
